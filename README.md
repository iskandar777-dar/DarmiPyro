## VPS/Locally deploy!
```console
 $ sudo apt-get update -y
 $ sudo apt-get upgrade
 $ git clone https://github.com/iskandar777-dar/DarmiPyro
 $ cd DarmiPyro
 $ python3 -m venv venv
 $ source venv/bin/act*
 $ pip3 install -U-r req*
 $ cp sample.env .env
```
